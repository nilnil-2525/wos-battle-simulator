wos_battlesimulator.html 計算ロジックまとめ

1. 基礎ステータス
- 兵種は shield / spear / bow。
- Tier ごとに attack / lethality / defense / hp の基礎値を持つ。
- 実ステータスは「基礎値 × (1 + 入力バフ% / 100)」で計算する。
- T11 弓の攻撃は常時 ×1.06。
- T11 盾が防御側の場合、防御は常時 ×1.06。

2. 総兵数
- 総兵数は shield.troops + spear.troops + bow.troops。
- 戦闘開始時に、味方・敵の小さい方の総兵数を minTotalTroops として固定する。
- 固定した minTotalTroops は全ターンのダメージ計算に使う。

3. 攻撃対象
- 通常ターゲットは「盾 → 槍 → 弓」の順で、生存している最初の兵種。
- T11 槍の「奇襲」が発動すると、そのターンの槍攻撃対象は弓に固定される。
- 弓がいない場合は空振りになる。

4. ターン進行
- 1 ターン内の攻撃順は、生存兵種の「盾 → 槍 → 弓」。
- 味方と敵が同じ優先度フェーズでそれぞれ攻撃する。
- 撃破数はフェーズ内で計算後、対象兵数から減算する。
- どちらかの総兵数が 0 になると戦闘終了。
- 最大ターン数は通常戦闘・モンテカルロともに 1000。

5. 基本ダメージ式
- baseDmg = (攻撃側攻撃 × 攻撃側殺傷) / (防御側防御 × 防御側 HP) / 100
- 通常撃破数の素値:
  baseDmg × sqrt(攻撃側兵数) × sqrt(開始時最小総兵数) × 兵種相性 × 通常スキル補正 / T7 盾軽減
- 追加撃破数の素値:
  baseDmg × sqrt(攻撃側兵数) × sqrt(開始時最小総兵数) × 兵種相性 × 追加スキル補正 / T7 盾軽減
- 最終撃破数:
  round(通常撃破 + 追加撃破)

6. 兵種相性
- 有利相性は ×1.1。
- 有利関係は、盾 → 槍、槍 → 弓、弓 → 盾。
- 槍が Tier7 以上の盾を攻撃する場合、追加で /1.1 軽減される。

7. 通常スキル補正
- 分子カテゴリ:
  DamageUp1 / DamageUp2 / DamageUp3 / NormalDamageUp / OppDefenseDown1 / OppDefenseDown2
- 分母カテゴリ:
  OppDamageDown1 / OppDamageDown2 / DefenseUp1 / DefenseUp2 / DefenseUp3 / DefenseUpS
- 通常Mod = 分子の各(1 + 値)の積 / 分母の各(1 + 値)の積。

8. 追加ダメージ補正
- ExtraDamageUp を追加倍率として扱う。
- 追加Modの分子:
  ExtraDamageUp合計 / DamageUp1 / DamageUp2 / DamageUp3 / OppDefenseDown1 / OppDefenseDown2
- 追加Modの分母:
  OppDamageDown1 / OppDamageDown2 / DefenseUp1 / DefenseUp2 / DefenseUp3 / DefenseUpS
- NormalDamageUp は追加ダメージには乗らない。

9. バフ対象判定
- 自軍バフは all / 対象兵種 / self に一致すると加算する。
- 敵側デバフは all_enemy / enemy_兵種 に一致すると加算する。
- spear_target や enemy_target は、記録済みの攻撃対象と一致した場合に加算する。
- 同カテゴリ内は加算、カテゴリ間は乗算する。

10. 永続スキル
- timing: always は 1 ターン目に remain: 999 として付与される。
- 以後、永続バフとして残る。

11. ターン周期スキル
- turn_3n / turn_4n / turn_5n は、有効ターン数がその倍数のとき発動する。
- スタンで行動不能になった場合、stunTurnOffset により周期判定がずれる。
- duration があるバフはターン終了時に remain を減らし、0 になると消える。

12. 槍の偶数回攻撃スキル
- 槍の攻撃回数を spearAttackCount で管理する。
- 偶数回攻撃時、spear_even_attack_instant はその攻撃の追加ダメージに即時加算される。
- 偶数回攻撃時、spear_even_attack_after は攻撃後に対象付きバフとして付与される。（奇襲発動時などは、実際に攻撃した部隊タイプを対象として引き継ぐ）

13. レネの槍追加攻撃
- spear_rene_timing は有効ターンが 3 以上かつ奇数のとき発動する。
- ExtraDamageUp として槍に追加倍率 2.00 を付与する。

14. ミアの確率スキル
- ミアI:
  2 ターン目以降、各フェーズ開始時（そのフェーズで行動可能な自部隊が生存している場合のみ）に 50% で発動。
  OppDefenseDown1 として敵被ダメ上昇を、そのフェーズの通常攻撃・ヘンドリック追加攻撃に適用する。
- ミアII:
  通常攻撃時に 50% で発動。
  ExtraDamageUp として追加ダメージを加算する。
  ヘンドリック全体攻撃時は発動しない。
- ミアIII:
  ターン開始時に 40% で発動。
  DefenseUp3 を 1 ターン付与する。

15. ソニヤIIIのスタン
- 槍攻撃時、有効ターンが 5 の倍数なら turn_5n_instant が発動する。
- 追加ダメージ倍率 2.50 を加算する。
- 攻撃対象を次の行動時に 1 回スタンさせる。
- スタンした部隊は攻撃をパスし、その後スタン解除される。

16. ヘンドリックIII
- 弓が攻撃した後、有効ターンが 3 の倍数なら発動する。
- 敵の生存している盾 / 槍 / 弓それぞれに追加狙撃する。
- 通常ダメージは 0。
- 追加ダメージ倍率は固定 0.40。
- ヘンドリック追加攻撃でも、フェーズ開始時に抽選済みのミアI効果は乗る。

17. ブラッドリー特殊処理
- 攻撃側の弓リーダーが bradley の場合、防御側の盾が攻撃を受けるとき HP を /1.25 する。
- 攻撃側の弓リーダーが bradley の場合、防御側の槍が攻撃を受けるとき HP を /1.30 する。
- 結果として防御側の受けるダメージが増える。
- DamageUpB カテゴリ自体はダメージ計算のカテゴリ集計には使われていない。

18. 無名の追加ダメージ耐性処理
- 防御側バフに無名があり、防御対象が盾で、追加ダメージがある場合に処理される。
- 追加ダメージ計算時の DefenseUp3 を -0.25 + 0.30 に置き換える。
- 通常ダメージ側には通常の DefenseUp3 が使われる。

19. T11 兵士スキル
- T11 弓:
  常時攻撃 ×1.06。
  通常攻撃時、10% で「連射」発動、追加倍率 +1.00。
  通常攻撃時、30% で「燃晶火薬」発動、追加倍率 +0.875。
- T11 槍:
  ターン開始時、敵弓が生存していれば 20% で「奇襲」、槍の対象を弓に固定。
  通常攻撃時、15% で「炎晶戦矛」、通常・追加の両方を ×2。
- T11 盾:
  常時防御 ×1.06。
  被攻撃時、37.5% で「烈晶盾」、受ける通常・追加の両方を /1.51。
- T11 槍の防御側:
  被攻撃時、15% で「熾烈領域」、受ける通常・追加の両方を /2。

20. 撃破数の減算
- 撃破数は Math.round() で丸める。
- 兵数は Math.max(0, 現在兵数 - 撃破数) で 0 未満にならない。
- ヘンドリックの全体追加攻撃は、通常攻撃の撃破処理とは別に即時で各対象から減算される。

21. モンテカルロ
- 1000 回シミュレーションする。
- 各回は現在の編成・兵数・バフ状態を初期値としてコピーする。
- ランダム発動を含めて最大 1000 ターン戦闘する。
- 結果として winner / allySurviving / enemySurviving を保存する。
- 勝利時残兵の平均・中央値・分散・標準偏差を計算する。

22. グレッグのスキル
- グレッグⅠ（スキル1）:
  ターン開始時に20%の確率で発動し、味方全部隊に殺傷バフ（DamageUp1）+40%を3ターン付与する。
- グレッグⅡ（スキル2）:
  各フェーズ開始時、敵が生存している場合に20%の確率で発動し、敵の殺傷低下デバフ（OppDamageDown1）-50%を2ターン付与する。
- グレッグⅢ（スキル3）:
  永続（always）で味方全部隊にHPバフ（DefenseUp1）+25%を付与する。

23. ゴードンのスキル
- ゴードンⅠ（スキル1）:
  槍兵の偶数回攻撃時に発動。そのターンの追加ダメージに即時加算（ExtraDamageUp）+0.75され、攻撃後に敵の対象部隊へ殺傷低下（OppDamageDown1）-15%を1ターン付与する。
- ゴードンⅡ（スキル2）:
  3の倍数ターンに発動。槍兵の与ダメ（DamageUp2）+20%および敵全体への殺傷低下デバフ（OppDamageDown1）-20%を1ターン付与する。
- ゴードンⅢ（スキル3）:
  4の倍数ターンに発動。敵盾兵の被ダメ上昇デバフ（OppDefenseDown1）+25%および敵弓兵の殺傷低下デバフ（OppDamageDown2）-25%を1ターン付与する。

24. エディスのスキル
- エディスⅠ（スキル1）:
  永続。弓兵の被ダメ低下（DefenseUp3）+20%および槍兵の与ダメ（DamageUp2）+20%を付与する。
- エディスⅡ（スキル2）:
  永続。盾兵の被ダメ低下（DefenseUp3）+20%を付与する。
- エディスⅢ（スキル3）:
  永続。全部隊のHP（DefenseUp1）+25%を付与する。

25. ジェロニモのスキル
- ジェロニモⅠ（スキル1）:
  永続。全部隊の殺傷（DamageUp1）+25%を付与する。
- ジェロニモⅡ（スキル2）:
  永続。全部隊の攻撃（DamageUp3）+25%を付与する。
- ジェロニモⅢ（スキル3）:
  4の倍数ターンに発動。全部隊の与ダメ（DamageUp2）+30%を2ターン付与する。

26. 暫定仕様・検証中の挙動（旧memo.txtより）
- 火晶盾スキルとミアⅠの抽選:
  各攻撃フェーズで 1 回のみ抽選し、通常攻撃と追加攻撃にまとめて適用する。火晶盾スキルは抽選回数が増えるもの（ソニヤ追加ダメ）と増えないもの（ゴードン追加ダメ）があるが、網羅できないためシミュレーター上では省略している。
- ガトのシールド:
  挙動詳細が未解明のため、暫定的に「敵与ダメ減30%」に近い補正値として処理している。駐屯側にガトがいる時にズレが生じる傾向があり、攻撃値または攻撃バフを参照している可能性があるが、ソユンによる攻撃UPの効果は適用されない。
- スキル重複枠と挙動の検証方法:
  火力系はinana氏のスキル種別表（熊罠の検証に基づく）を参照。耐久系やゴードン、レネの遠征1などは熊罠で検証できないため、仮実装から実機戦闘結果との一致を確認して判別している。確率スキル（ミアや火晶兵スキル）は発動タイミングが戦闘レポートから直接判別できないため、ゲーム内の説明文を基に暫定実装している。
- T8兵士のスキル:
  テスト用兵士のため、連射/奇襲などの確率要素は未実装。ただしパッシブスキル（盾から槍への被ダメージ10%減少など）は実装済み。
